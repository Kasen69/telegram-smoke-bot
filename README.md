\# Telegram Smoke Bot



Простий Telegram-бот для підрахунку перекурів.



\## Команди



\- /start

\- /smoke

\- /info

\- /top



\## Запуск



Встановити залежності:



```bash

pip install -r requirements.txt

```



Встановити змінну середовища:



```bash

TOKEN=ВАШ\_ТОКЕН

```



Запустити:



```bash

python bot.py

```
